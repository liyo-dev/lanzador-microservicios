Guía rápida de navegación

Esta aplicación centraliza la gestión de microservicios y acceso a portales. A continuación, una breve descripción de cada sección:

  • ⚙️ Configuración: Define rutas de repositorios, puertos, variables de entorno y parámetros globales para tus servicios Angular y Spring.
  • 🚀 Lanzador: Arranca o detén múltiples microservicios simultáneamente, consulta logs en tiempo real y gestiona el estado de cada servicio.
  • 👥 Usuarios Portal: Administra credenciales de acceso organizadas por entorno para acceder directamente a los portales sin necesidad de recordar contraseñas.
  • 🔌 Gestión de Puertos: Encuentra y libera puertos ocupados que puedan estar bloqueando el arranque de tus servicios.
  • ✅ Tareas: Lleva tu propia lista de tareas con prioridades, fechas, etiquetas y filtros, y prepárate la daily con lo de ayer y lo de hoy. Persistencia local automática.

Configuración

Establece rutas de repositorios, puertos de servicio y variables de entorno necesarias para el correcto funcionamiento de tus microservicios.

Lanzador

Inicia o detén múltiples servicios de forma simultánea y supervisa el estado de ejecución y logs en tiempo real.

Gestión de Usuarios

Administra credenciales de acceso organizadas por entorno para facilitar la autenticación en diferentes portales.

Gestión de Puertos

Encuentra y libera puertos ocupados. Identifica procesos que bloquean el arranque de servicios y termínalos fácilmente.

Tareas

Tu lista personal con prioridades, fechas, etiquetas y filtros, y la vista Daily para ver lo que hiciste ayer y lo que toca hoy.
